Postpartum Medicaid Access Barrier Index

Development and internal validation of a county-level screening index

Project site for a county-level screening index of overlapping administrative, clinical, and access-support barriers relevant to postpartum Medicaid access.

Research question: Where may postpartum Medicaid populations face overlapping administrative, clinical, and contextual access barriers, and which counties does the full index surface beyond a screen focused only on Medicaid office and hospital-based obstetric care gaps?

1. Dashboard 2. Findings Brief 3. Technical Report

What This Project Contributes

  • IndexA transparent county-level score combining Medicaid office availability, hospital-based obstetric care status, ACS access indicators, and rural-urban context.
  • Main FindingThe full index identifies high-burden counties where contextual barriers elevate concern beyond what infrastructure gaps alone would show.
  • How To ReviewUse the dashboard for visual exploration, the findings brief for a concise stakeholder summary, and the technical manuscript for methods and validation.
